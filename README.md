# 🚀 SkillForge Pro

### Complete Registration & Dashboard Web Application

---

## 📌 About Project

SkillForge Pro is a modern, fully responsive web application that allows users to register with complete details (personal, education, skills, projects) and manage their profile through an interactive dashboard. All data is stored locally using LocalStorage API.

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- LocalStorage API
- Font Awesome Icons
- Google Fonts

---

## ✨ Key Features

**Registration Form (Multi-Step)**
- Profile photo upload with live preview
- Personal details (Name, Email, Mobile, DOB, Gender)
- Complete education (10th, 12th, Graduation with board/stream/percentage)
- Skills & hobbies multi-select tags
- Dynamic project addition (Name, Description, Dates, Role)
- Resume upload (PDF/DOC)
- Smooth step navigation with animations

**User Dashboard**
- Profile photo & welcome message
- Personal information card
- Education details card
- Skills & hobbies badges
- Projects list display
- Resume filename display
- Edit profile & Logout functionality

**UI/UX Features**
- Dark/Light theme toggle with localStorage persistence
- Fully responsive (Mobile, Tablet, Desktop)
- Smooth animations & transitions
- Clean minimalist layout
- Sidebar navigation

## 🚀 How to Run

1. Download all files (index.html & script.js)
2. Save them in the same folder
3. Open index.html in any modern browser
4. Start registering!

> No server required! Works completely offline.

---

## 💾 Data Storage

All user data is saved in browser's LocalStorage. Data persists even after browser restart. Click Logout to clear data.
